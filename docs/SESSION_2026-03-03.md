## Update (same date): G2.manual unified manual-cycle summary contract

### Что сделано

* Добавлен новый helper `scripts/check_gateway_sla_manual_cycle_summary.py` с CLI:
  * `--runs-dir`,
  * `--preflight-summary-json`, `--readiness-summary-json`, `--governance-summary-json`, `--progress-summary-json`, `--transition-summary-json`,
  * `--policy report_only|fail_if_blocked`,
  * `--summary-json`.
* Зафиксирован machine-readable контракт `gateway_sla_manual_cycle_summary_v1`:
  * required source: `preflight`,
  * optional sources: `readiness/governance/progress/transition`,
  * для каждого source публикуется `status=present|missing|invalid`,
  * `decision` копируется из preflight (`accounted_dispatch_allowed`, `decision_status`, `next_accounted_dispatch_at_utc`, `reason_codes`),
  * `observed` агрегирует `readiness_status`, `governance_decision_status`, progress/transition ключевые поля.
* Добавлены regression tests `tests/test_check_gateway_sla_manual_cycle_summary.py` (happy/block/fail-policy/missing optional/missing-invalid preflight/CLI help).
* Обновлены source-of-truth docs: `docs/RUNBOOK.md`, `docs/DECISIONS.md`, `TODO.md`.
