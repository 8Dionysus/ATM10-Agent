# SESSION_2026-02-20

Краткий конспект ключевых результатов сессии от **2026-02-20**.

## Что сделано

* Для retrieval внедрён двухэтапный pipeline (`candidate-k` + optional reranker).
* Для `qwen3` добавлен runtime switch `torch|openvino` и device `AUTO|CPU|GPU|NPU`.
* Для Windows добавлен wrapper: `scripts/run_qwen3_openvino.ps1`.
* В ingest FTB Quests расширен SNBT extraction:
  * поддерживаются quoted и unquoted значения ключей
  * ключи: `id`, `type`, `dimension`, `structure`, `filename`, `group`, `title`, `subtitle`, `description`.
* First-stage ranking в retrieval улучшен:
  * field-weighted scoring (`title/text/tags`)
  * stopword filtering для снижения ложных совпадений по служебным словам.
