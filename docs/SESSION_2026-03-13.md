## Update (same date): G2 single-cycle operator helper

### Что добавлено

* Добавлен новый local helper `scripts/run_gateway_sla_operating_cycle.py` с контрактом `gateway_sla_operating_cycle_v1`.
* Helper делает один `G2` operating cycle end-to-end:
  * сначала читает latest `readiness/governance/progress/transition/remediation/integrity/cadence`
  * определяет, свежи ли required summaries за текущие UTC-сутки
  * если latest snapshot уже свежий, переиспользует его без нового `manual_nightly`
  * если latest stale/missing/invalid, запускает fallback в фиксированном порядке `manual_nightly -> cadence -> remediation -> integrity`
* Helper пишет:
  * machine-readable summary `runs/nightly-gateway-sla-operating-cycle/operating_cycle_summary.json`
  * краткий human-facing brief `runs/nightly-gateway-sla-operating-cycle/triage_brief.md`
* Добавлены regression tests:
